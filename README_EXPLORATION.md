## 🛠️ How to Use These Materials

### For Writing Your Page

1. **Prepare** (1 hour)
   - Read START_HERE.md + WRITER_GUIDE.md + PAGE_REFERENCE_MAP.md
   - Bookmark context-index.md for reference

2. **Research** (2-3 hours)
   - Read relevant sections of context-index.md
   - Skim source files listed in PAGE_REFERENCE_MAP.md
   - Study docs/getting-started.md and docs/architecture.md for style

3. **Write** (4-8 hours)
   - Follow WRITER_GUIDE.md templates
   - Use context-index.md to look up facts (Ctrl+F)
   - Create code examples
   - Add cross-references to other pages

4. **Review** (2-3 hours)
   - Check PAGE_REFERENCE_MAP.md "What to Include"
   - Use WRITER_GUIDE.md quality checklist
   - Verify all file paths in context/skyvern/
   - Verify all examples are correct

**Total: 3-5 days per page**

### For Reviewing a Page

Use the quality checklist from WRITER_GUIDE.md:
- All required sections included? ✓
- Code examples tested and correct? ✓
- File paths verified in source code? ✓
- Follows WRITER_GUIDE format? ✓
- Tone matches other pages? ✓
- Cross-references accurate? ✓
